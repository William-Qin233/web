# 1. 排序和搜索是什么
- 排序：把某个乱序的数组变成升序或降序的数组；
- 搜索：找出数组中某个元素的小标；
# 常用方法
JS中的排序： 数组的 sort 方法
JS 中的搜索：数组的 indeOf 方法
## 排序算法：

- 冒泡排序
- 选择排序
- 插入排序
- 归并排序
- 快速排序
## 搜索算法

- 顺序搜索
- 二分搜索 
# 冒泡排序
## 思路
比较所有的相邻元素，如果第一个比第二个大，则交换它们；
一轮比较下来，可以保证最后一个数是最大的。
执行 n - 1 轮，就可以完成排序了；
```javascript
// bubbleSort.js

Array.prototype.bubbleSort = function() {
  for(let i = 0; i < this.length - 1; i++) {
    for(let j = 0; j < this.length - 1 - i; i++){
      if(this[j] > this[j + 1]){
        const tem = this[j]
        this[j] = this[j + 1]
        this[j + 1] = tem;
      }
    }
  } 
}

const arr = [5, 4, 3, 2, 1]
arr.bubbleSort();
```
